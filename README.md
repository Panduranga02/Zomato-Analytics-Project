#  Zomato Analytics Project

##  Business Problem
Analyze restaurant data to identify trends in ratings, pricing, and customer preferences to help improve decision-making for food delivery platforms.

##  Project Overview
This project analyzes Zomato restaurant data to extract insights on restaurants, ratings, cuisines, pricing, and sales trends. Interactive dashboards were created using Power BI, Tableau, and Excel.

##  Tools & Technologies
- Power BI
- Tableau
- Excel
- SQL
- Python (optional for preprocessing)

##  Dataset
- Restaurant data (CSV files)
- Country & Currency mapping

##  Key Insights
- Total Restaurants: 9551+
- Average Rating: ~2.89
- Majority of restaurants do not offer table booking (~91%)
- Online delivery available in ~22% of restaurants
- New Delhi has the highest number of restaurants

##  Features
- Rating-wise restaurant distribution
- Price bucket analysis
- Monthly & yearly trends
- Online delivery & table booking insights
- City-wise and country-wise analysis

##  Learnings
- Data cleaning and transformation
- Dashboard design principles
- Business insights extraction
